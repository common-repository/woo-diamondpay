== Description ==
 
Accept Payment through credit cards / debit cards using DiamondPay Payment Gateway.

This plugin extends WooCommerce allowing user to take payments directly on your store via DiamondPay API.

To Visit DiamondPay Official URL  <a href="http://www.diamondbank.com/business/cash-management-services/payments/diamondpay/">Click Here</a>

<h4>Others WooCommerce Payment module</h4>

== Installation ==
 
1) Upload 'woo-diamondPay' folder to the '/wp-content/plugins/' directory
2) Activate 'woo-diamondPay' form wp plugin option in admin area
3) Plugin will appear in the menu bar of WP Dashboard
4) set the plugin settings from WooCommerce checkout setting
